# 📒 AUTONOTES
POSTECH CSED 과목 자료들을 gemini flash 2.5를 이용해 페이지별 자동 해설한 레포지터리입니다.
노트는 자유롭게 활용하시고 코드 보완 해주시면 감사하겠습니다.
다른 과목도 자유롭게 추가해주세요

## 💾 과목 리스트
1. CSED226 데이터분석 입문 (한욱신 교수님)

## ⚙️ 설치 방법

1. 이 저장소를 클론합니다.
2. 아래 명령어를 실행하여 필요한 라이브러리를 설치합니다.
   ```bash
   pip install -r requirements.txt
   ```
3. .env 에 `GEMINI_API_KEY=YOURKEYHERE` 설정합니다
4. 코드 상에서 PDF 경로를 수정하고 실행합니다.
